Cyber Essentials Readiness

Five-control self-test against Cyber Essentials v3.3 ('Danzell', in force 28 April 2026). Indicative only.

1. Boundary firewalls and internet gateways

Default admin passwords changed; admin interfaces are not exposed to the public internet.

2. Secure configuration

Unused services removed; auto-run disabled; default credentials changed on every device.

3. User access control + MFA

Unique accounts per user, least-privilege admin, MFA on all cloud and admin accounts.

4. Malware protection

Anti-malware enabled and updating on every in-scope device, with real-time scanning.

5. Security update management (14-day SLA)

High/critical patches applied within 14 days of vendor release on all in-scope devices.

Pass Thresholds
