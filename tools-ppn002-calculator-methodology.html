How the PPN 002 Calculator sizes the 10% minimum

A full account of PPN 002, the procurement framework that sets the 10% social value minimum, the TOMs measures, and the proxy values we use. Defensible enough to cite in a bid response.

1. PPN 002: what changed and when

The locked wording: PPN 002 (effective 24 February 2025) sets the minimum social value weighting at 10%. PPN 002 supersedes PPN 06/20 for new procurements from that date. Existing contracts already procured under PPN 06/20 continue under that framework; only new procurements move to PPN 002.

Both frameworks share the same theoretical foundation: a Social Value Model with five themes and eight policy outcomes, scored using the National TOMs Framework. PPN 002 simplifies the routing and locks the 10% minimum.

2. The 10% minimum

The 10% figure is the minimum weighting on social value within the overall award criteria. It is a floor, not a ceiling. Buying organisations frequently set higher weightings — 15%, 20%, occasionally 25% — where the contract has a clear social-value angle. The Calculator assumes the floor by default and flags the higher cases.

The 10% minimum applies to in-scope public sector procurements. It does not apply to: most local authority below-threshold procurements, framework call-offs where the framework already scored social value at award, or procurements expressly carved out by procurement policy notes.

3. The TOMs Framework

TOMs (Themes, Outcomes, Measures) is the de-facto scoring rubric. The National TOMs Framework lists Measures grouped by Outcome and Theme. Each Measure carries a quantifiable indicator (e.g. "Number of FTE local employees hired") and a proxy value (e.g. social value of one full-year local hire).

The Calculator uses the National TOMs Framework as the canonical list, with our internal cross-mapping to PPN 002's eight outcomes. The full CrowMark platform extends this with bespoke TOMs libraries for sector-specific contracts.

4. Oxford Social Value Bank (SVB) proxy values

Proxy values are stored in our database (table toms_measures_library) and updated when the Oxford SVB publishes a new annual edition. We never hardcode proxy values in product code — that is a project rule recorded in CLAUDE.md.

The Calculator uses the most recent published Oxford SVB. The current edition cited in the tool is 2023–24. Proxy values are converted to the contract's currency and time horizon; multi-year contracts are scored across the contract life and discounted as appropriate.

5. NPV discount and time horizon

For multi-year contracts, the Calculator discounts forward-year social value cashflows. The locked wording: NPV calculations use the HM Treasury Green Book rate of 3.5%. This is the same rate CrowAgent Core uses for retrofit-NPV; it is the appropriate UK public-sector discount rate.

Energy and labour cost escalations are NOT applied to social value cashflows by default; only the time-value-of-money discount applies. The full CrowMark platform allows custom escalation if your buyer specifies one.

6. From floor to scoring

The Calculator returns a defensible scoring window. A bid responding to a 10% weighting cannot win on social value alone, but a credible response to the 10% threshold materially improves overall score — especially in close calls. The scoring window distinguishes:

  • The minimum credible response (matches the floor; defensible).
  • The competitive response (clear delivery plan; matches sector norm).
  • The differentiated response (additional commitments; supports a "best in class" pitch).

The Calculator is deliberately neutral on which response level you should target — that is a commercial decision, not a calculator output.

7. What the Calculator does not do

It does not write the bid narrative. It does not track post-award delivery. It does not produce branded PDF outputs. Those capabilities live in CrowMark. The Calculator exists to answer one question fast: what is a defensible 10% response, in pounds, against this contract?
