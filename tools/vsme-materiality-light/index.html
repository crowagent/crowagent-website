Methodology & sources

Why you can trust this path recommendation

  • Path logic is sourced from EFRAG VSME (December 2024), the voluntary SME-friendly standard.
  • Module B (Basic) covers 11 disclosures applicable to all SMEs; Module C (Comprehensive) adds 9 more for SMEs facing value-chain pressure.
  • Size bands come from EU Commission Recommendation 2003/361 (micro, small, medium).
  • B11 (own workforce) remediation flag highlights when headcount-by-gender, training, and pay-gap data isn't yet tracked.
