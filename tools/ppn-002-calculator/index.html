Verification

Methodology & Sources

How the 10% floor is calculated.

Statutory Basis

Calculation is based on the Cabinet Office Procurement Policy Note (PPN 002/24) issued in February 2025.

Weighting Floor

PPN 002 mandates a minimum 10% social value weighting for all in-scope central government contracts.

TOMs Framework

Tool uses the Themes, Outcomes, and Measures (TOMs) framework to align qualitative missions with quantitative scoring.

Proxy Values

Proxy values are derived from the Oxford Social Value Bank (SVB) and GSS benchmarks for SMEs.
