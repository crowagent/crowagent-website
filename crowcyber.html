03 / The Scope

The five NCSC
technical controls.

Control 1

Firewalls

Boundary firewalls and internet gateways configured to block unapproved services.

Control 2

Configuration

Devices hardened from default builds; unnecessary functionality removed.

Control 3

Updates

Supported software, patched within the v3.3 SLA for high/critical vulnerabilities.

Control 4

Malware

Anti-malware or application allow-listing across in-scope devices.

Control 5

Access Control

Least-privilege accounts, MFA on cloud admin (mandatory under v3.3 Danzell).
