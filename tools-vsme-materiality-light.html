How the screen works

From three answers to a reporting path in three steps

Take the size + pressure inputs

Organisation size (EU 2003/361 bands), exposure to a B-listed sector or CSRD-customer pressure, and Module B11 own-workforce data readiness. Three answers and you have your screen.

Apply the VSME path test

Medium SMEs and any SME under value-chain pressure default to Comprehensive (Module B + Module C, 20 disclosures). Micro/small SMEs without pressure get Basic (Module B, 11 disclosures).

Render the path + remediation flag

Recommended path, total disclosure count, B11 readiness flag where data is missing, and EFRAG VSME (December 2024) citations. A baseline you can defend to a listed-company customer.

FAQ

Common questions

What is VSME?

EFRAG VSME (2024) is the SME-friendly path for voluntary sustainability reporting.

Is this full double materiality?

No. This is a Light screen for a fast first pass. CrowESG (Coming Q3 2026) runs the full assessment.

Do I need this if I'm in CSRD scope?

No. CSRD requires full ESRS double-materiality. VSME is for entities outside CSRD scope.
