Methodology in one paragraph

What the screen does

The screen walks through six VSME-aligned steps: business model, value chain, stakeholders, impact identification, financial-effects identification, and materiality scoring. Output is a portable materiality matrix. The full CrowESG platform (Coming Q3 2026) runs full double materiality with stakeholder input and cross-framework metric mapping.

FAQ

Common questions

What is VSME?

EFRAG VSME (2024) is the SME-friendly path for voluntary sustainability reporting.

Is this full double materiality?

No — this is a Light screen for a fast first pass. CrowESG (Coming Q3 2026) runs the full assessment.

Do I need this if I'm in CSRD scope?

No — CSRD requires full ESRS double-materiality. VSME is for entities outside CSRD scope.
