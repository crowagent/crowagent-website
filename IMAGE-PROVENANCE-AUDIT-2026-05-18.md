## 5. Going forward — process gate

Per founder rule:

1. Every new photo entered into `Assets/photos/` MUST be accompanied by a manifest entry that includes `pageUrl` (Unsplash/NASA/Pexels source URL) + `photographer` + `cdnId`.
2. A pre-push grep can enforce this — proposed:

```bash
# Pre-push check: every JPG in Assets/photos/ must be covered by manifest.
node -e "
const fs = require('fs');
const m = JSON.parse(fs.readFileSync('Assets/photos/_manifest.json'));
const covered = new Set(m.flatMap(e => [e.jpg, e.webp]));
const files = fs.readdirSync('Assets/photos/').filter(f => /\\.(jpg|jpeg|webp)\$/.test(f) && !f.startsWith('hero-earth-') && !f.startsWith('hero-premium-earth-'));
const orphans = files.filter(f => !covered.has(f));
if (orphans.length) { console.error('orphan photos (no manifest):', orphans); process.exit(1); }
"
```

(NASA hero images are exempt — they're covered by a separate audit entry in this document.)
